Different Ways of Using Angular Directive in HTML

In this example we will learn how to use Angular Directive in 6 different ways within our HTML

HTML:

                

                    <ol>
                        <!-- 1. -->
                        <li>
                            <my-first-directive></my-first-directive>
                        </li>
                        <!-- 2. -->
                        <li>
                            <div my-first-directive></div>
                        </li>
                        <!-- 3. -->
                        <li>
                            <div data-my-first-directive></div>
                        </li>
                        <!-- 4. -->
                        <li>
                            <div x-my-first-directive></div>
                        </li>
                        <!-- 5. -->
                        <li>
                            <div my:first:directive></div>
                        </li>
                        <!-- 6. -->
                        <li>
                            <div my_first_directive></div>
                        </li>
                    </ol>

JavaScript:

                

                    /////////////
                    //Angular App
                    ////////////
                    var myApp = angular.module("myApp", []);


                    /////////////
                    //Angular Controller
                    ////////////
                    myApp.controller('Controller1',[Controller1]);

                    function Controller1(){

                    }

                    /////////////
                    //Angular Directive
                    ////////////
                    myApp.directive('myFirstDirective',[MyFirstDirective]);

                    function MyFirstDirective(){
                        return{
                              template:'
Hello From My First Directive
'
                        }
                    }

  9. Class Form

    For this to work, you have to add 'restrict' property to the directive 
            return{
            restrict:'EAC',
                ...
        }
    'E' means use as element, 'A' means use as attribute, 'C' means use as class.
    E & A are default values. If you want to use directive as a class you have to use 'restrict' property and mention 'C' in it(and also EA unless you not only want it for directive to be used as class only)
